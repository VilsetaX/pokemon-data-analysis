# Pokémon GO PvP Analysis

This project explores the relationship between Pokémon base stats and PvP performance in Pokémon GO.

Using base stat data and Great League rankings, the analysis investigates which stat profiles are common among top-ranked Pokémon.

## Data Sources

Base stats dataset derived from Pokémon GO Game Master data.

PvP rankings sourced from PvPoke.

https://pvpoke.com

## Questions

* Do top PvP Pokémon prioritize defense over attack?
* Is stamina a common characteristic among top performers?
* What stat distributions appear in the Great League meta?
